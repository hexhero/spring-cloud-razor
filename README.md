# Spring-Cloud-Yoyo (:construction:项目正在施工中)

## 项目结构
```
project
  | api-common          # 存放实体和封装类
  | business-*          # 业务模块
  | serve-discover      # 服务发现
  | serve-gateway       # 网关服务
  | serve-oauth2        # OAuth2认证中心
```
